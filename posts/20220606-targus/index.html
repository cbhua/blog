开箱 Targus 转接坞

2022.06.06

买了 Mac mini 之后发现外接三个显示器不行啦，最后搞得不得不再买一个转接坞。

书接上回，买了新的 M1 芯片的 Mac mini 回来之后发现 M1 芯片只限制连接两个屏幕，从第三个屏幕开始就自动 Mirror 第一个屏幕，这个还是系统级别的操作，我们还什么都做不了。如果想要第三个屏幕怎么办呢？要加钱上 M1 pro 或者 M1 Ultra 芯片，真有你的啊，库克。

所以最后又买了一个 Targus 的 Display Link 支持的转换坞才解决了这个问题。虽然价格挺贵的，但是买来用了一阵子之后发现贵的转换坞也确实有贵的道理，和我之前用过的比较廉价的转换坞根本没法比。想到曾经我还用绿联的转换坞烧掉了一个笔记本的主板。所以说绿联转接坞烧主板的事情是真的，而且还不好找维修，很难确定原因。

说起来转接坞，我到现在为止买过三个转接坞，最开始本科的时候为了省钱，就买了那个绿联的转接坞，价格也就不到 200 块，提供的接口还是挺全的。这个转接坞其实支撑了大概半年的时间，后面的故事就是把我笔记本的主板烧了。

其实烧得不是特别彻底，不是说笔记本直接无法启动或者冒烟、着火、爆炸、核爆、产生黑洞这种，而是电脑开始不定期随机蓝屏，所以说是定时炸弹。这可比不同的损坏让人眼前一黑（字面意思）多了。

我买的这个型号的 Targus 转接坞背后有两个 USB-A 接口、一个网线、两个 HDMI 和一个 Thunderbolt 3 的 Type C，正面有一个 USB-A、一个不是 Thunderbolt 的 Type C 还有一个极其先进的 3.5mm 耳机接口。

我的接法是这样的：把 Mac mini 本体的 HDMI 先用上，接上主屏幕，然后用一根 C to C 的线连接 Mac mini 的 Thunderbolt 3 C 口和 Targus 的 Thunderbolt 3 口，因为要作为视频输出所以带宽需求还是挺高的，有一个 Thunderbolt 3 接口就完全足够两个 2K 屏幕的输出带宽了，官方说最高可以供一个 5K 屏幕加上一个 4K 屏幕，一个人就是一个审片组。之后剩下的两个屏幕接到 Targus 转接坞上面，通过 Display Link 输出虚拟信号。这样整体接线就完成了。

Targus 是一个主动供电的转接坞，需要有一个专用的充电线来供电，这让我本来桌面就不充裕的充电插口雪上加霜。好在最后插线板扩展插线板给我提供了广阔的拓展空间。因为都是小功率电器所以感觉这样套娃应该问题不是很大。

插上去之后，在 Mac 上面安装了 Display Link 的驱动，顺利点亮了三个屏幕，令人感动！使用起来也感觉不到是软件渲染的桌面，没有延迟和卡顿，和原生的屏幕连接质量一致。

用久了之后发现 Targus 会微微发热，去搜了搜是正常情况。随后就测试了一下几个接口的速度，都是能达到最新的 USB-A 3.0 的满速，平常 Copy 个文件啊视频素材啊什么的都没问题。而且这样就不用每次插硬盘都得摸 Mac mini 的屁股，直接插在转接坞的正面接口就行。

总体来说 Targus 虽然不会让我感觉到性价比，但是确实能体会到一分价钱一分货，还是不错的购买体验。最主要的是它能解决第三个屏幕显示的这个核心生产力问题，所以不用它还不行。
